# Исследование диагностической эффективности методов лучевой диагностики в выявлении метастазов печени

## Цель исследования

Оценка диагностической эффективности различных методов лучевой диагностики в выявлении метастазов печени.

## Участники исследования

1. **Главный исследователь (монитор)** — 1 рентгенолог с опытом работы в лучевой диагностике печени не менее 5 лет
2. **Группа А** — врачи-рентгенологи с опытом работы в лучевой диагностике печени не менее 5 лет
3. **Группа Б** — 2 врача-рентгенолога с опытом просмотра КТ ангиографии не менее 1 года
4. **Группа В** — рентгенхирурги с опытом работы в печени не менее 5 лет
5. **Эксперт** — специалист, создающий и обновляющий эталонную мастер-модель

## Изучаемые модальности

Пациент при соответствии критериям включения поступает на операцию. За 3 дня до операции ему выполняются:
- **КТ** — компьютерная томография с внутривенным контрастированием
- **МРТ** — магнитно-резонансная томография с внутривенным контрастированием
- **КТ-АГ** — КТ ангиография
- **МРТ-АГ** — МРТ ангиография
- **ПДКТ-АГ** — позитронно-эмиссионная компьютерная томография ангиография

Дополнительно:
- **КТ-архив** — предыдущие КТ, выполненные до химиотерапии (может быть несколько исследований)

Каждая модальность представлена **одним** исследованием (кроме КТ-архива). Сегментация непосредственно на КТ-архив не выполняется. Однако сегментация КТ и КТ-АГ осуществляется в двух вариантах: изолированной серии и совместно с КТ архивом.

## Основная концепция: мастер-модель

**Мастер-модель** — эталонная сегментация всех выявленных очагов у пациента, где каждый очаг расположен в отдельной зоне интереса (ROI) с уникальным номером.

Основой мастер-модели является первая КТ пациента с завершённой сегментацией. Координатная сетка (число пикселей, размеры пикселей, точка отсчёта) определяется эталонной серией из первичной оценки качества и остаётся неизменной.

Мастер-модель создаётся один раз и дополняется новыми очагами по мере их обнаружения на исследованиях других модальностей. Каждый новый очаг получает уникальный номер.

Для каждой модальности (МРТ, КТ-АГ, МРТ-АГ, ПДКТ-АГ) создаётся **проекция** мастер-модели — перенос очагов из координатного пространства КТ в координатное пространство этой модальности.

## Этапы исследования

### 1. Оценка качества (ручное)

Выполняется врачами-рентгенологами группы А (2 независимые оценки)

На этом этапе определяется:
- Пригодность исследования для дальнейшей работы
- Тип модальности (КТ, МРТ, КТ-АГ, МРТ-АГ, ПДКТ-АГ, КТ-архив)
- **Эталонная серия** — серия с наименьшим числом артефактов и наибольшим числом анатомических ориентиров, на которой будет выполняться разметка очагов

Поиск очагов осуществляется на всех доступных сериях, но разметка выполняется только на эталонной серии.

### 2. Анонимизация (автоматически)

Выполняется автоматически при прохождении проверки качества. Все персональные данные пациента заменяются на анонимные идентификаторы.

### 3. Сегментация очагов (ручное)

**Кто выполняет:** врачи соответствующей специализации в зависимости от модальности

- КТ → врачи со специализацией в КТ
- МРТ → врачи со специализацией в МРТ
- КТ-АГ → врачи со специализацией в КТ-ангиографии
- МРТ-АГ → врачи со специализацией в МРТ
- ПДКТ-АГ → врачи со специализацией в ПДКТ

Каждое исследование размечается двумя врачами независимо

**Доступная информация:**

- Для КТ (стандартный контекст): только текущее исследование
- Для КТ (расширенный контекст): текущее исследование + все КТ-архивы пациента
- Для других модальностей: только текущее исследование

**Процесс:**

1. Врач просматривает анонимизированное исследование в OHIF viewer
2. Загружает эталонную серию в 3D Slicer
3. Размечает очаги в трёх категориях: метастазы, неопределённые образования, доброкачественные образования

Особенности этапа, в том что оценка КТ с данными архива проводится всегда после того как им проведена оценка одиночного исследования. Если врач допускается к оценке нескольких модальностей, то модальности ему демонстрируются в последовательности от потенциально менее чувствительного метода до более чувствительного: КТ, МРТ, КТ-АГ, МРТ-АГ, ПДКТ-АГ, КТ с архивом. Между оценками одним врачем разных модальностей должно пройти не менее 2х дней, либо врачем должно быть оценено не менее 10 других исследований.

Сегментация выполняется на эталонной серии, но врач может просматривать все доступные серии.

### 4. Первичное построение мастер-модели

Производится экспертом после завершения первой КТ-сегментации с данными архива (любого из врачей). Координатная сетка для модели определяется эталонной серией для КТ. Каждый очаг из сегментации получает отдельную метку соответстующую его уникальному номеру. 

### 5. Построение проекции мастер-модели (ручное)

Производится экспертом для каждой новой модальности (МРТ, КТ-АГ, МРТ-АГ, ПДКТ-АГ).

В 3D Slicer эксперт просматривает эталонные серии целевой модальности и КТ с нанесенной на ней мастер-моделью. По очереди наносит ROI с мастер-модели на координатное пространство серии целевой модальности. 
Если очаг не виден на целевой модальности то ROI наносится исходя из анатомических ориентиров. Для определения расположения очага на исследоании целевой модальности необязательно использовать только эталонную серию, также можно просматриать другие серии этого исследоания.

При сохранении разметка проекции мастер модели валидируется на предмет наличия всех очагов и того что никакой из очагов не представлен двумя отдельно лежащими ROI.

### 6. Сравнение проекции и сегментации (автоматически)

Сопоставление сегментации серии каждой из модальностей с мастер-моделью осуществляется через проекции. Производится автоматическое попиксельное сравнение сегментации и проекции. Совпадениями считается пересечения ROI более чем на 5% их объёма. 

Таким образом очаги могут быть классифицированы на: выявленные, пропущенные, дополнительно выявленные. К дополнительно выявленным очагам относятся очаги которые были выявлены при сегментации, но отсутствуют на мастер-модели.

При наличии дополнительно выявленных очагов исследование переходит к этапу обновления мастер-модели.

При наличии пропущенных очагов врач, который осуществлял сегментацию переходит к этапу "Пересмотр".

Этот этап выполняется для каждой сегментации независимо (если 2 врача выполнили 2 сегментации то проводится 2 сравнения). 

### 7. Пересмотр

Выполняет врач, чья сегментация расходится с проекцией мастер-модели. Врач получает изображение эталонной серии в 3D Slicer с нанесенными очагми из проекции, которые врач пропустил (каждый под своим номером).  Задача врача отнести очаг к одной из четырех категорий: метастазы, неясные, доброкачественные, невидимые.

Эта задача всегда выполняется врачем, в сегментации которого выявлены расхождения. Врач не знает по данным какой модальности очаг был выявлен и как был охарактеризоан, ему известна только зона интереса на проекции мастер-модели. Если в результате обновления мастер-модели были выявлены новые расхождения, то задача инвалидируется и её требуется решить заново, но уже для новых очагов. Все результаты этого этапа сохраняются в единственный файл вне зависимости от числа итераций.

При сохранении врачу доступна только классификация очагов из проекции мастер-модели, выделение новых очагов не доступно.

### 8. Обновление мастер-модели

При обнаружении на этапе сравнения проекции и сегментации дополнительно выявленных очагов эксперту дается задача обновления мастер-модели. 

В Slicer 3D выдается мастер модель с эталонной серией КТ, проекция модели на модальность, также на проекцию нанесены дополнительно выявленные очаги (выделены дополнительным цветом). Эксперт должен перенести эти дополнительные очаги с проекции на мастер-модель.

Нумерация нанесенных на мастер-модель очагов осуществляется автоматически. Для избежания состояния гонки и дублирования работы по переносу вновь выявленного очага в каждый момент времени активной может быть только одна задача обновления, даже если дополнительные очаги были выявлены на разных модальностях.

После обновления мастер-модели этапы нанесения проекции, сравнение проекций и сегментаций иналидируются и проводятся заново. 
Создаются новые задачи для экспертов на редактирование проекций.для врачей

Координатная сетка мастер-модели не меняется

Обновление не удаляет существующие очаги, только добавляет новые. Номера очагов на мастер модели являются иммутабельными, при сохранении проверяется, что ранее присутствующие на мастер-модели очаги не изменили номер.


**Проблема:** мастер-модель может быть обновлена в процессе создания проекции. Соответственно проекция будет сохранена после сохранения мастер модели.

**Решение:**
- При создании проекции сохраняется контрольная сумма (хэш) используемой мастер-модели
- При завершении проекции хэш сравнивается с текущим файлом мастер-модели
- При несовпадении проекция инвалидируется и создаётся новая задача
- Потери работы не происходит — проверка только при завершении задачи

## Пример последовательности действий

1. **КТ:** первичная оценка качества → определение эталонной серии → сегментация врачом → создание мастер-модели → проекция на ту же серию (копия)
2. **МРТ:** первичная оценка качества → определение эталонной серии → сегментация врачом → создание проекции мастер-модели экспертом → автоматическое сравнение
3. **Расхождение:** обнаружен 1 очаг на сегментации МРТ, отсутствующий в проекции (false positive) → создаётся задача обновления мастер-модели
4. **Обновление:** эксперт добавляет новый очаг на мастер-модель → все существующие проекции инвалидируются
5. **Повторные проекции:** эксперты заново размечают проекции для всех модальностей
6. **Пересмотр:** для врачей, чьи сегментации расходились с обновлённой проекцией, создаются задачи пересмотра
7. **КТ-АГ:** первичная оценка → сегментация → проекция → сравнение → расхождений не выявлено
8. **ПДКТ-АГ:** аналогичный цикл → расхождений не выявлено
